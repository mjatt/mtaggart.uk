Commands

Command Usage Description
8ball +8ball [question] Gives you a random response to your query
avatar +avatar [@user] Returns the user's avatar. Leave blank to get your own.
ban +ban [@user][reason] Bans the selected user and sends a private message showing the reason they were banned.
born +born [@user] Returns the time and date the user was created. Leave blank to get your own.
createtag +createtag [tagName][tag content] Creates a custom command for a server. See +tag for how to activate your tag.
gif +gif [content] Searches gifs for your content and returns a random gif.
help +help Directs you to this page
id +id [@user] Returns the user id of a user. Leave blank to get your own.
invite +invite Returns the link to invite the bot to a server and the link to my Discord.
kick +kick [@user][reason] Kicks the mentioned user and sends a private message with the reasoning.
levels +levels [on|off] Turns the levelling system on or off for a guild. Default set to off.
log +log Creates a #moderation channel for the guild. Set up to work with +warn.
love +love [content] Gives you a love rating out of 100 between you and the content.
nation +nation [nation] Returns some nation information from nationstates.
ping +ping Shows you how long it takes for the bot to reply.
play +play [youtubeURL] Plays a YouTube video. This also sends a message with playback controls.
pokemon +pokemon [pokemonName|gif] [pokemonName] Returns a pokemon picture and id. Use [gif] to return the gif of the pokemon.
purge +purge [number < 100] Removes the selected number of messages from a channel.
region +region [region] Returns some region information from nationstates.
remindme +remindme [time in minutes][content] Pings the user after the time allocated with the message. If the bot goes offline; it will not save the remindme.
roll +roll Rolls a dice.
say +say [content] Makes the bot echo your message.
server +server Returns some information about the server.
stab +stab [content] Simulates stabbing the content.
stats +stats Returns some stats about the bot.
tag +tag [tagName] Returns the content of the created tag.
waifu +waifu [content] Rates how good the content is as a waifu out of 10.
warn +warn [@user][reason] Adds a message to #moderation if it exists which warns the user for breaking the rules.
weather +weather [location] Returns some weather information from the location.
xkcd +xkcd [comic number][random] Returns an xkcd comic. Leave blank for the latest, else use random or the comic number.
